# AI-Generated Evidence Detection Benchmark

A NOBLE-funded research project evaluating how well AI-generated image detection tools perform on law enforcement content (bodycam, surveillance, evidence photos).

## Getting Started

**Read the student guide:** [ashleyscruse.github.io/ai-generated-image-detection](https://ashleyscruse.github.io/ai-generated-image-detection/)

The guide covers cloning the repo, setting up your environment, using Git, and accessing TACC.

## Your Notebook

Your personal notebook is in `notebooks/portfolios/`. Open it, run the cells, and fill in the "YOUR TURN" sections. Everything you need is inside.

## Quick Setup

```bash
git clone https://github.com/ashleyscruse/ai-generated-image-detection.git
cd ai-generated-image-detection
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
